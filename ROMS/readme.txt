Esta carpeta contiene un fichero binario para grabar las ROM del sistema OMEGA con 2 paginas distintas.
Una de las páginas tiene el sistema configurado con la ROM MSX2+ original y la segunda incluye varias modificaciones:
- Configuración regional internacional.
- Símbolo de contrabarra en lugar del yen.
- Soporte para fmpac (en caso de no estar instalado algunos juegos pierden el sonido).
Con este sistema, conectando un interruptor a JP1 de la placa base se puede conmutar entre ambas opciones, la configuración
regional es inmediata, el soporte de FMPAC requiere reiniciar.

Se incluye el script de creación y los ficheros necesarios para modificarlo al gusto.
